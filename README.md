# Hit-and-Blow-2

ブラウザ上で遊べる 4 桁版のヒット＆ブローです。オプションを切り替えて難易度や遊び方を自由に調整できます。

## はじめに

このアプリケーションはCodex Cloudを利用して作成されました。  
Zenn記事用に作成されたアプリケーションです。  
https://zenn.dev/beagle/articles/bc6ef88dd68615

## 遊び方

1. CPU が 4 桁のシークレットコードを作成します。
2. プレイヤーは推測した 4 桁のコードを入力して判定を行います。
3. 判定結果は以下の通りです。
   - **Hit**: 位置と文字の両方が一致
   - **Blow**: 文字は含まれるが位置が違う
4. すべての桁を当てるまで何度でも推測できます。

## オプション

ゲーム画面上部の「オプション」でルールを変更できます。設定を変更すると自動的に新しいゲームが開始されます。

- **数字の重複**
  - デフォルトは「重複なし」で、同じ文字を複数回使うことはできません。
  - 「重複あり」に切り替えると、シークレットコードにも推測にも同じ文字を何度でも使用できます。
  - 重複なしモードでは、入力欄でも同じ文字を 2 回以上入力できないようになっています。
- **利用する文字の種類**
  - 使用する文字の種類を 6〜16 の範囲で選択できます。
  - 6 を選ぶと「0〜5」、10 を選ぶと「0〜9」、11 以上では数字に続いて「A〜F」が順番に追加されます。
  - どの設定でもシークレットコードは 4 桁です。重複なしを選択する場合は 4 文字以上の種類を選択してください。

## 技術情報

- HTML/CSS/JavaScript で構成された単一ページアプリケーションです。
- `gameLogic.js` にゲームロジックをまとめ、UI 操作は `script.js` が担当しています。
- `npm test` でゲームロジックの単体テストを実行できます。
